# 일기당천 (一騎當千) — 삼국 서바이버

> 한 명의 장수로 수천의 적병을 베어내는 브라우저 뱀서라이크.
> 뱀파이어 서바이버의 중독 루프 × 삼국무쌍의 쾌감 × Three.js 화려한 이펙트.

- 플랫폼: 브라우저 (데스크톱 + 모바일), Three.js + Vite + TypeScript
- 배포: https://midagedev.github.io/threesur/
- 런 길이: 10분 (사망 시 즉시 결과). 목표: "한 판만 더" 중독 루프.

---

## 1. 코어 루프

1. 장수 선택 → 10분 생존전 시작
2. 이동만 조작 (WASD/조이스틱). 무기는 전부 자동 발동
3. 적 처치 → 혼백 젬 드랍 → 레벨업 → 3택 업그레이드 (무기/패시브)
4. 1.5분마다 엘리트(A급 장수), 3/6분 중간보스, 9분 최종보스 여포
5. 킬 콤보 → 무쌍 게이지 → Space로 무쌍난무 (화면 청소 궁극기)
6. 사망/클리어 → 골드 정산 → 메타 상점(영구 강화) → 재도전

## 2. 조작

| 입력 | 동작 |
|---|---|
| WASD / 화살표 | 이동 |

### 패시브 (최대 6슬롯, Lv5)
적토마 편자(이속+8%/lv) · 현철갑주(피해감소 8%/lv) · 병법서(경험치+8%/lv) · 백주(체력+10%/lv, 재생) · 전국옥새(골드+10%/lv) · 향로(픽업반경+15%/lv) · 호부(쿨다운-5%/lv) · 주작깃발(투사체+1, Lv3 희귀)

### 진화 (무기 Lv8 + 지정 패시브 → 엘리트 보물상자)
- 청룡언월도 + 현철갑주 → **참마검**: 회전 참격파를 발사
- 백우선 + 병법서 → **팔문금쇄진**: 관통 부적 폭풍
- 화계 + 백주 → **적벽업화**: 전진하는 화염 해일
- 천뢰 + 호부 → **천벌뇌정**: 적간 연쇄 번개
- 제갈연노 + 주작깃발 → **원융노**: 전방위 연사

## 6. 성장/경제

- 혼백 젬: 파랑(1) · 초록(5) · 빨강(20), 픽업 반경 기본 2m, 젬은 자석 흡인 연출
- 레벨: `nextXp = 5 + level×10`, 레벨업 시 일시정지 + 3택 카드 (+골드 리롤 1회 50G)
- 킬 콤보: 3초 내 킬 유지, 100/500/1000 돌파 시 "百人斬!" 배너 + XP 보너스, 콤보가 무쌍 게이지 가속
- 무쌍 게이지: 킬 +1%, 피격 +3%. 발동 시 5초: 적 시간 0.3×, 무적, 장수별 난무, 종료 충격파
- 골드 → 메타 상점 (localStorage): 공격/체력/이속/반경/쿨다운/시작레벨/부활 각 5단계, 비용 `200×2^n`. 여포 해금 5000G
- 도감: 처치한 보스/엘리트 기록, 최고 기록(생존시간·킬수·레벨) 표시

## 7. 비주얼 디렉션 — "Fable 데모급"

- **무대**: 밤의 전장. 어두운 남흑색 지면(절차 생성 수묵 노이즈 텍스처 + 은은한 격자), `FogExp2`, 화면 곳곳 떠다니는 불씨·반딧불 파티클
- **카메라**: PerspectiveCamera FOV 40, 상방 55° 내려보기, 플레이어 부드러운 추적, 트라우마 기반 셰이크 + 킬 펀치줌
- **렌더링**: ACESFilmicToneMapping, UnrealBloomPass(strength≈0.9, threshold≈0.6) — 모든 이펙트는 HDR emissive로 블룸을 태운다
- **스프라이트**: NearestFilter, 세워진 빌보드 쿼드(살짝 기울임), 발밑 타원 블롭 그림자, 걷기 4프레임 8fps
- **적 렌더**: 시트별 InstancedMesh + 커스텀 셰이더 (인스턴스 속성: uv 프레임, 피격 플래시, 스케일, 틴트) — 600+ 마리 60fps
- **타격감**: 피격 화이트 플래시, 넉백, 엘리트/보스 히트스탑 40ms, 사망 파티클 버스트 + 디졸브
- **무기 이펙트**: 애디티브 슬래시 아크(리본 트레일), 지그재그 라이트닝+글로우, GPU 화염 파티클, 확장 충격파 링, 유도 부적 트레일
- **무쌍 연출**: 시간 슬로우, 비네트+채도 시프트, 방사형 스피드라인, 대형 한자 "無雙" 플래시, 종료 대형 충격파
- **데미지 숫자**: 캔버스 스프라이트 풀, 크리티컬은 크고 노랗게
- **UI 톤**: 먹빛 배경 + 금색 헤어라인, 한자 병기 (예: "무쌍 無雙"), 세리프 명조 계열

## 8. 사운드 (WebAudio 프로시저럴, 외부 파일 없음)

타격(짧은 노이즈+저역 펀치), 젬(펜타토닉 상승 핑), 레벨업(공, 심벌), 무쌍(대북+슬로우 웅웅), 보스 등장(뿔피리), 클리어(팡파레). BGM: 펜타토닉 스케일 절차 루프(고쟁 느낌 플럭), 음소거 토글.

## 9. 기술 아키텍처

```
src/
  main.ts              # 부트스트랩, 씬 상태머신 (Title→Select→Run→Result)
  core/   loop.ts input.ts rng.ts events.ts save.ts audio.ts
  gfx/    renderer.ts(postfx) camera.ts atlas.ts sprites.ts(instanced)
          particles.ts(GPU) effects.ts(slash/lightning/ring/trail) ground.ts damageText.ts
  game/   run.ts player.ts enemies.ts(pool+grid) spawner.ts collision.ts
          weapons/*.ts pickups.ts levelup.ts musou.ts boss.ts combo.ts stats.ts
  data/   heroes.ts weapons.ts passives.ts waves.ts upgrades.ts spriteManifest.ts
  ui/     hud.ts screens.ts(title/select/result/shop) cards.ts joystick.ts
public/assets/sprites/  # sgrade/apriority/soldiers/variants 시트 + manifest.json
```

- 적/젬/파티클: 고정 크기 풀 + freelist, 프레임당 할당 0 목표
- 충돌: 균일 그리드(셀 2m) 스페이셜 해시 — 적↔적 분리(soft), 적↔플레이어, 무기 판정은 원/부채꼴/캡슐 수학 판정
- 게임 시간과 렌더 시간 분리(무쌍 슬로우는 적 dt에만 적용), 고정 dt 60Hz 시뮬 + 보간 없이 단순 가변 dt(클램프 33ms)도 허용
- 저장: localStorage key `threesur-save-v1` (골드, 메타 레벨, 해금, 도감, 최고기록)
- 성능 목표: 데스크톱 60fps @ 적 600, 모바일 30fps+ (DPR 클램프 2, 모바일 블룸 해상도 절반)

## 10. 재미 극대화 요소 (Phase 4)

### 전장 이벤트 (런에 변주와 서사를 주는 랜덤 사건, 화면 상단 경고 배너 + 전용 사운드)
- **보급 마차** (2~8분 사이 랜덤 2회): 발광하는 마차(runner 스프라이트 + 수레 연출 + 금색 트레일)가 화면을 가로질러 도주. 처치 시 골드 대량 + 보물상자. 놓치면 사라짐 — 추격의 긴장감.
- **황건 대군 러시** (4분·7분): 30초간 한 방향에서 대군 돌진(일직선 웨이브). 처치 밀도가 높아 콤보/젬 파밍 기회.
- **유성우** (6분 이후 랜덤): 15초간 하늘에서 화염 운석 낙하(경고 데칼 → 착탄 폭발). 적도 플레이어도 피해 — 지형 리스크.

### 전장 오브젝트 (맵에 산재, 상호작용)
- **화약통**: 무기로 터뜨리면 대폭발(반경 4m, 연쇄 유폭). 스폰율 낮게, 위기 탈출 수단.
- **만두 수레**: 파괴 시 만두 드랍(HP 30% 회복). 5분 이후 희소.
- **군신 사당**: 접촉 시 30초 버프 1종(공격력 +30% / 이속 +25% / 무쌍 게이지 2배 충전). 무너진 석상 연출, 사용 후 재사용 불가.

### 저주받은 유물 (risk-reward 레벨업 카드, 10% 확률로 등장, 보라색 테두리)
- 예: "칠성검 — 공격력 +40%, 최대 체력 -30%" / "동작대 금령 — 골드 +100%, 이속 -15%" / "우선(牛扇) — 쿨다운 -25%, 픽업 반경 -50%". 최대 2개 보유.

### 장수별 무쌍난무 전부 차별화 (DESIGN 3절 표대로: 관우 회전참·장비 포효 스턴·제갈량 낙뢰 폭풍·황충 화살 폭풍·여포 조작형 돌진)

### 콤보 피버: 콤보 200+ 유지 시 "피버" — 화면 가장자리 금색 불꽃, XP 획득 1.5배, 전용 사운드 레이어. 끊기면 해제.

### 업적 + 칭호 (localStorage, 도감 탭에 표시)
- 예: 첫 승리(천하의 주인), 여포 처치(비장군 참살), 천인참(千人斬), 노히트 3분(금강불괴), 진화 5종 달성(전설의 대장장이), 누적 만 킬(백전노장) 등 12종 내외. 달성 시 토스트 알림. 칭호는 결과/공유 카드에 표시.

### 무한 모드: 10분 승리 후 "계속 싸운다" 선택 → 스케일 무한 증가(11분+ 매분 HP 1.35^n), 기록은 별도 리더보드(최장 생존).

## 12. 스프라이트 시트 규격 (public/assets/sprites/manifest.json)

- 셀 48×64px, 콘텐츠 32×48 bottom-center 앵커
- 행 = 방향 [south, west, east, north], 열 = 프레임 [idle, step1, idle2, step2]
- 캐릭터 i 블록 = 열 `i*4 .. i*4+3` (sgrade 20명 / apriority 16명 / soldiers 10종)
- soldiers-variants.png: 168×64 셀 그리드, 4×4 블록 단위. manifest의 `variants[archetype] = [{c,r,v}]` — 블록 픽셀 원점 = `(c*192, r*256)`. 아키타입당 24색 변형
- walk 애니메이션 프레임 순서: idle→step1→idle2→step2 루프
